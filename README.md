# Sukhov-journal

Сервис статей, с возможностью их озвучки.


## Tech Stack
- Go (Echo)
- Python (FastApi)
- PostgreSQL (Gorm)
- Docker / Docker Compose

## Архитектура

Проект состоит из двух сервисов, запущенных через Docker Compose:

1. **Articles Service (Go)** — основной бэкенд:
   - хранит статьи,
   - принимает запросы от клиента,
   - управляет пользователями и авторизацией,
   - отправляет текст на озвучивание.

2. **TTS/STT Service (Python)** — сервис обработки речи: # в разработке
   - принимает текст от Go-сервиса,
   - генерирует аудио (Text-to-Speech),
   - при необходимости выполняет распознавание речи (Speech-to-Text).

Сервисы общаются между собой по HTTP внутри Docker-сети. (связывание сервисов в разработке, но проект уже можно запустить)
